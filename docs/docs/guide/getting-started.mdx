## Installation

You can use **SomaUI** in two ways:

1. **Via CLI (recommended)** â€“ generate projects and components with ease.
2. **As a package** â€“ import components directly from `@somaui/ui`.

## CLI Usage

SomaUI comes with a handy CLI to generate components and scaffold projects.
