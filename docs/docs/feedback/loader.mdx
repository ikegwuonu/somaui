# Loader

A loader for displaying loading state of a page or a section.

```jsx
import { Loader } from '@somaui/ui/loader';
```

<div className="mb-20 md:mb-28" />

## Default

The default style of the **Loader** component.

<Paper>
  <Loader />
</Paper>

```jsx
import { Loader } from '@somaui/ui/loader';

export default function App() {
  return <Loader />;
}
```

<div className="mb-16" />

## Variants

You can use different variants of our **Loader** component.

<Paper>
  <div className="flex w-full items-center justify-around gap-3">
    <Loader className="h-auto w-7" />
    <Loader variant="spinner" className="h-auto w-7" />
    <Loader variant="pulse" className="h-auto w-7" />
    <Loader variant="threeDot" className="h-auto w-7" />
  </div>
</Paper>

```jsx
import { Loader } from '@somaui/ui/loader';

export default function App() {
  return (
    <>
      <Loader />
      <Loader variant="spinner" />
      <Loader variant="pulse" />
      <Loader variant="threeDot" />
    </>
  );
}
```
